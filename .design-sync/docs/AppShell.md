# AppShell

The full MixJam application window — `Header` (with its live theme selector) +
the tracker/home body + `Footer`, composed and wired exactly as the real
`App.tsx`. Use this to preview a whole page and switch between all 8 themes
live from the header dropdown, rather than reviewing one component in isolation.

The `Default` story opens in Emerald with a working theme selector; the
`Theme*` stories open directly on a specific theme.
